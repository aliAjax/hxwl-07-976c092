📝 手动测试步骤

以下步骤需要人工操作验证，用于补充自动化测试覆盖不到的场景。

  1. 首次访问 & SW 注册
    清理浏览器缓存后首次访问页面，检查 DevTools → Application → Service Workers 中 SW 状态为 activated。
  2. 离线加载核心页面
    在 DevTools → Network 勾选 "Offline"，刷新页面，确认仍能正常显示（或显示 offline.html）。
  3. 断网保存检查记录
    开启离线模式，在主应用中添加/编辑一条检查记录，确认数据保存到本地且同步队列数量 +1。
  4. 同步队列展示
    查看同步队列 UI，确认队列中显示正确的操作类型、摘要和状态（待同步）。
  5. 恢复在线后自动同步
    取消 "Offline" 勾选恢复在线，观察队列操作是否自动变为"已同步"状态，数量递减。
  6. 缓存清理功能
    点击"清理缓存"按钮，检查 DevTools → Application → Cache Storage 中缓存被清除，SW 被注销。
  7. 后台同步（Background Sync）
    离线时添加操作，然后关闭网络恢复在线，确认即使刷新页面操作仍被同步。
  8. 跨标签页同步
    打开两个标签页，在一个标签页中添加操作，确认另一个标签页的队列状态同步更新。
